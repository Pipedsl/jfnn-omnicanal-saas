# Registro de Bugs y Capturas de Pantalla

Carpeta destinada a almacenar capturas de pantalla, imágenes de referencia y registros visuales de bugs del sistema que debamos ir resolviendo.

## Cómo registrar un Bug:
1. Sube la foto o captura de pantalla a este directorio (`documentacion/bug_reports/`).
2. Dale un nombre claro (ej. `bug_error_despacho_20260619.png`).
3. Si lo deseas, puedes documentar el detalle o registrar el backlog en un archivo `.md` de esta misma carpeta o en el backlog del proyecto.
